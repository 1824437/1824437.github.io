&& 逻辑符号

前面的命令正确, 后面的继续执行

|| 逻辑符号

前面的命令无效, 后面的执行。

; 顺序执行

一个挨一个执行

& 后台工作

单一个& 符号，且放在完整指令列的最后端，即表示将该指令列放入后台中工作。

| 管道 (pipeline)

pipeline 是 UNIX 系统，基础且重要的观念。连结上个指令的标准输出，做为下个指令的标准输入。

> 输出到文件
ls >1.txt

>> 追加到文件
cat a.txt>>b.txt

-

伪设备

  • /dev/null

    cat a.txt >/dev/null 黑洞，屏幕上不会显示任何信息。
